# MIRA
## *Um olhar sobre o Brasil que você ainda não viu!*

A MIRA é uma plataforma de streaming focada exclusivamente em produções nacionais, criada para valorizar o cinema brasileiro e torná-lo acessível para todos.
Nosso objetivo é aproximar o público de quem cria, oferecendo uma experiência inclusiva, intuitiva e totalmente imersiva.

O site apresenta a proposta da MIRA, sua identidade visual, funcionalidades e pilares de acessibilidade — audiodescrição, LIBRAS, legendas descritivas e curadoria acessível — reforçando nosso compromisso com inclusão, cultura e representatividade.

Mais do que uma plataforma, a MIRA é um movimento que celebra histórias brasileiras, dá espaço para criadores independentes e democratiza o acesso ao audiovisual no país
